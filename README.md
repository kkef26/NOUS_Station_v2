# NOUS Station v2

Living-system command interface — NST.19.7 r3.

Three-surface triple (Chat / Boardroom / Project), four drawers (Pulse / Factory / Fleet / Signals), dual-theme (navy+teal dark + warm-paper light), keyboard-driven navigation.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to `main` → Vercel auto-deploys.

## Audit

```bash
npm run audit
```

Checks for banned design patterns per NST.19.7 r3 §8.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| ⌘1 | Chat |
| ⌘2 | Boardroom |
| ⌘3 | Project |
| ⌘P | Pulse drawer |
| ⌘F | Factory drawer |
| ⌘L | Fleet drawer |
| ⌘S | Signals drawer |
| ⌘K | Composer |
| ⌘, | Settings |
| ⌘⇧T | Theme toggle |
| Esc | Close overlay |
